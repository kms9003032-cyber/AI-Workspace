# Report 1 - Chess Piece Automatic Recognition and Arrangement Robot System

# 1. 프로젝트 목표

본 프로젝트의 목표는 체스판을 인식하고, 체스말의 상태와 위치를 판단한 뒤, 실제 로봇이 체스말을 자동으로 이동 및 정리하는 시스템을 제작하는 것이다.

시스템의 주요 기능은 다음과 같다.

- 체스판 인식
- 체스말 종류 인식
- 체스말 상태 판단
- 좌표 변환
- 체스 엔진 연동
- Arduino 기반 실제 모터 제어
- 기물 자동 정리

본 프로젝트는 단순한 체스 AI 구현이 아니라,
"Vision → 판단 → 물리 제어"가 연결된 실제 피지컬 AI 시스템 구현을 목표로 한다.

---

# 2. 프로젝트 배경 및 구조 변경 이유

초기에는 실제 건설 현장의 타워크레인과 유사한 구조를 구현하려 하였다.

하지만 실제 구현 과정에서 다음과 같은 문제점이 발생하였다.

- 구조 안정성 부족
- 흔들림 증가
- 정밀 제어 어려움
- 제작 비용 증가
- 공간 제약 발생

따라서 현재는:

"타워크레인의 기본 원리는 유지하되,
보다 안정적이고 정밀 제어가 가능한 구조"

로 방향을 수정하였다.
