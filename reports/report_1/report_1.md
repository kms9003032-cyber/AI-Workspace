# 1. 프로젝트 개요

최근 로봇 기술과 인공지능 기술의 발전으로 다양한 자동화 시스템이 개발되고 있으며, 체스와 같은 전략 게임에서도 로봇과 AI를 결합한 시스템이 연구되고 있다.

그러나 기존 체스 로봇 시스템은 주로 기물 이동 기능에 집중되어 있으며, 실제 사용 환경에서 발생할 수 있는 다양한 변수—예를 들어 기물의 쓰러짐, 잘못된 위치 배치, 체스판 외부로 벗어난 기물—에 대한 대응이 부족한 한계를 가진다.

본 프로젝트는 이러한 문제를 해결하기 위해:

- 체스 기물 위치 인식
- 상태 판단
- 자동 복구
- 자동 정리

까지 수행 가능한 지능형 체스 로봇 시스템 설계를 목표로 한다.

---

# 2. 구조 변경 배경

초기에는 실제 건설 현장의 타워크레인과 유사한 구조를 구현하는 것을 목표로 하였다.

타워크레인 구조는:
- 넓은 작업 범위
- 효율적인 위치 제어
- 평면 이동 구조

라는 특징을 가진다.

하지만 실제 구현 과정에서:

- 구조적 안정성 부족
- 진동 및 흔들림 문제
- 정밀 제어 어려움
- 제작 비용 증가
- 공간 제약

문제가 발생하였다.

따라서 현재는:

> 타워크레인의 기본 원리는 유지하되,
> 보다 안정적이고 정밀 제어가 가능한 구조

로 방향을 수정하였다.
