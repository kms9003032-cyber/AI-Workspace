# Chess Engine Code

## 목적

상부 카메라가 인식한 체스판 변화 데이터를 바탕으로  
사용자의 수를 판단하고,

체스 엔진을 통해 컴퓨터의 대응 수를 계산한 뒤,

실제 로봇 좌표 변환 단계로 넘기는 코드 관리 폴더이다.

---

# 포함 코드

## camera.py

상부 카메라를 통해 체스판 전체를 촬영하고,
각 칸의 변화 여부를 감지한다.

### 역할

- 체스판 4개 모서리 지정
- 체스판 격자 생성
- 각 칸의 기물 존재 여부를 0/1로 판단
- 사용자의 이동 전후 상태 비교
- 변화된 체스 좌표 추출

---

## game.py

체스 게임 엔진을 담당한다.

### 역할

- camera.py에서 받은 사용자 이동 입력 처리
- Stockfish 또는 체스 엔진 라이브러리 연동
- 현재 체스판 상태 저장
- 컴퓨터의 대응 수 계산
- 대응 수를 체스 좌표 형식으로 출력

---

## translation.py

체스 좌표를 실제 로봇 좌표로 변환한다.

### 역할

- e2, e4 같은 체스 좌표를 실제 mm 좌표로 변환
- 잡을 말의 위치 계산
- 말을 놓을 위치 계산
- Arduino로 전달할 이동 데이터 생성 준비

---

# 현재 핵심 연결 흐름

```text
camera.py
↓
game.py
↓
translation.py
↓
Arduino
↓
Motor Control
```

---

# 현재 우려점

- 각 코드의 입력/출력 형식이 완전히 통일되지 않았을 가능성 존재
- 상부 카메라 인식이 조명 변화에 민감함
- 실제 체스판 좌표와 로봇 좌표의 오차 검증 필요
- Arduino 명령 형식이 아직 최종 확정되지 않음
