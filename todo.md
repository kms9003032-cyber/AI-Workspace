# Current Bottleneck  
Vision 시스템과 좌표 변환 결과가 제어(로봇 암 컨트롤)로 제대로 연동되지 않음

# Why It Matters  
Vision에서 판별한 체스말 위치가 실제 로봇 암의 동작 좌표로 정확히 전달, 변환되어야 전체 로봇 체스 시스템이 유기적으로 작동함. 만약 변환 혹은 연동이 정확하지 않으면, 그리퍼가 잘못된 위치를 집거나 놓을 가능성이 큼. 시스템 완성도를 위해 이 흐름의 신뢰성이 최우선임.

# Today's One Action  
Vision에서 인식한 체스 좌표(픽셀/행열)를 로봇 암 좌표계(실제 x, y, z)로 변환하는 한 건의 end-to-end 테스트(실제로 팔을 움직여서 원하는 칸을 정확히 짚는지 확인)를 수행하라.

# Expected Result  
Vision→좌표 변환→로봇 암 제어 전체 연동 플로우의 문제(오프셋, 좌표 뒤틀림, 소프트웨어 버그 등)를 빠르게 발견할 수 있고, 전체 연결성 및 신뢰성 검증이 이루어짐.
